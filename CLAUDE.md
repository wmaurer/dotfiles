# CLAUDE.md

## Documentation rules

When adding or modifying a managed tool:
- Always update the **Managed tools** section in `README.md`.
- For simple configs, add a one-line description inline (e.g. `- **ToolName** — brief description`).
- For tools with complex setup or explanation, create a dedicated doc in `docs/` and link to it (e.g. `- **[ToolName](docs/toolname.md)** — brief description`).
